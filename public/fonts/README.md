# Fonts Directory

## Adding Sherika Font

To add the Sherika font to this project:

1. **Download the font files:**
   - For personal use: Download from [FontSpace](https://www.fontspace.com/sherika-font-f145304)
   - For commercial use: Purchase license from [MyFonts](https://www.myfonts.com/collections/sherika-font-seniors-studio)

2. **Required font files:**
   Place the following font files in this directory:
   - `Sherika-Regular.woff2`
   - `Sherika-Medium.woff2`
   - `Sherika-SemiBold.woff2`
   - `Sherika-Bold.woff2`
   - `Sherika-Regular.woff` (fallback)
   - `Sherika-Medium.woff` (fallback)
   - `Sherika-SemiBold.woff` (fallback)
   - `Sherika-Bold.woff` (fallback)

3. **Font weights available:**
   - Regular (400)
   - Medium (500)
   - SemiBold (600)
   - Bold (700)

4. **After adding the font files:**
   The font will be automatically loaded and available throughout the application.
